# Kanban: Optimizing Flow

## 1. Visualize Workflow

- Use a Kanban board (To Do, In Progress, Review, Done).

## 2. Limit WIP (Work in Progress)

- Set strict limits on how many items can be in each column to prevent bottlenecks.

## 3. Manage Flow

- Focus on the speed and quality of work movement across the board.

## 4. Make Process Policies Explicit

- Clearly define what "Done" means for each stage.

## 5. Continuous Improvement

- Regularly review cycle time and lead time to identify areas for optimization.
