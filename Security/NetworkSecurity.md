# Network Security Fundamentals

## 1. Principle of Least Privilege

Only allow the minimum necessary network traffic to and from your servers.

## 2. Firewalls

Use firewalls to control incoming and outgoing network traffic based on predefined security rules.

## 3. SSH Best Practices

Avoid using passwords for SSH access. Instead, use SSH keys and keep your keys secure.

## 4. SSL/TLS

Enable SSL/TLS for all network communications to encrypt data in transit and verify the identity of the server.

## 5. Regular Audits

Regularly audit your network security configuration and monitor for potential vulnerabilities.
