# OWASP Top 10 Overview

## 1. Description

The OWASP Top 10 is a list of the most critical web application security risks.

## 2. Broken Access Control

Broken access control occurs when an authenticated user is able to access resources or perform actions that they should not be authorized to.

## 3. Cryptographic Failures

Cryptographic failures occur when sensitive data is not properly protected, such as through encryption or access control.

## 4. Injection

Injection attacks occur when untrusted data is sent to an interpreter as part of a command or query.

## 5. Insecure Design

Insecure design is a broad category that includes failures in requirements, design, and architecture.
