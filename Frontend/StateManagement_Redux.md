# State Management with Redux Toolkit

## 1. Opinionated and Efficient

Redux Toolkit is the official, opinionated, batteries-included toolset for efficient Redux development.

## 2. Slices

Use `createSlice` to define your state, reducers, and actions in a single place.

## 3. Store Setup

Use `configureStore` to wrap `createStore` with sensible defaults and middleware like Redux Thunk.

## 4. Hooks

Use `useSelector` to read data from the store and `useDispatch` to dispatch actions.

## 5. Immer Integration

Redux Toolkit uses Immer internally, allowing you to write "mutative" logic that is immutable under the hood.
