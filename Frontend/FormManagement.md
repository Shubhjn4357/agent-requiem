# Modern Form Management (React Hook Form)

## 1. Uncontrolled Components

- Use `register` for high-performance forms with minimal re-renders.

## 2. Validation with Zod

- Integrate `zodResolver` to share validation logic between frontend and backend.

## 3. Schema-Driven Forms

- Define the form structure in a central `schemas/` directory to ensure consistency.

## 4. Error Handling

- Use `<ErrorMessage />` or accessible ARIA attributes to display validation errors.

## 5. Form State

- Access `isSubmitting`, `isValid`, and `dirtyFields` to manage UI states (e.g., disabling buttons).
