# AI Prompt Engineering: Advanced Techniques

## 1. Few-Shot Prompting

- Provide several examples of inputs and desired outputs in the prompt.

## 2. Chain-of-Thought

- Encourage the model to "explain its reasoning step-by-step" to improve accuracy on complex tasks.

## 3. Delimiter Usage

- Use clear delimiters (e.g., `###`, `---`, `"""`) to separate different sections of the prompt.

## 4. Temperature Control

- Use low temperature (e.g., `0.2`) for factual/logical tasks and high temperature (e.g., `0.8`) for creative tasks.

## 5. Iterative Refinement

- Treat prompt engineering as an experimental process. Monitor outputs and adjust the prompt based on observed failures.
