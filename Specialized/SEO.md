# SEO Optimization & Best Practices

## Metadata

1. **Title Tags**: Unique, descriptive titles for every page (< 60 chars).
2. **Meta Descriptions**: Compelling summaries (< 160 chars).
3. **Canonical Tags**: Use to prevent duplicate content issues.
4. **Open Graph (OG)**: Define `og:title`, `og:image`, and `og:description` for social sharing.

## HTML Semantics

- Use `<header>`, `<main>`, `<footer>`, `<section>`, `<article>`, and `<aside>` correctly.
- Single `<h1>` per page.
- Sequential heading order (`h1` -> `h2` -> `h3`).
- `alt` tags for every image.
